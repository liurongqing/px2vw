## px2vw

一个 px 单位转成 vw 单位的 VSCode 插件

![演示图](./screenshots/px2vw.gif)

## 安装
1. 通过命令行安装

    ```shell
    ext install px2vw
    ```

2. 直接在 vscode 编辑上插件上查找安装


## 使用

- 自动提示
- 快捷键 `ctrl+p` `ctrl+v`

## 支持语言

- html
- css
- less
- sass
- scss
- stylus

## 配置

- px2rem.width 设计宽度
- px2rem.toFixedNum 保留小数
